# Ryan Heejoon Byun academic homepage

This is a minimal static academic homepage for GitHub Pages.

## Deploy on GitHub Pages

1. Create or open the repository named `ryanheejoonbyun.github.io`.
2. Put these files in the root of the repository:
   - `index.html`
   - `style.css`
   - `assets/Ryan_Heejoon_Byun_CV.pdf`
3. Commit and push to the `main` branch.
4. In GitHub, go to **Settings > Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select branch `main` and folder `/root`, then save.
7. The site should appear at `https://ryanheejoonbyun.github.io`.

## Update the CV

Replace `assets/Ryan_Heejoon_Byun_CV.pdf` with the newest PDF and keep the filename the same.

## Optional edits

- Add a profile photo later by placing it in `assets/` and editing `index.html`.
- Add paper links once the preprint is posted.
- Add Google Scholar or GitHub links once they are polished.
